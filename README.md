# 🚢 Titanic — Previsão de Sobrevivência

Projeto de Machine Learning para prever a sobrevivência de passageiros do Titanic.

## 📋 Descrição
Análise completa do dataset do Titanic utilizando técnicas de Data Science,
desde a exploração dos dados até a construção de um modelo preditivo.

## 🎯 Resultado
- Acurácia do modelo: **81%**
- Algoritmo utilizado: Regressão Logística

## 🔍 Principais Insights
- Mulheres tiveram taxa de sobrevivência de 74% vs 19% dos homens
- Passageiros de 1ª classe sobreviveram mais (63%) que os de 3ª (24%)
- Sexo foi a variável mais importante para o modelo

## 🛠️ Tecnologias utilizadas
- Python
- Pandas
- Numpy
- Matplotlib
- Seaborn
- Scikit-learn

## 📊 Etapas do projeto
1. Carregamento e exploração dos dados
2. Limpeza e tratamento de nulos
3. Visualizações e análise estatística
4. Preparação para Machine Learning
5. Treinamento e avaliação do modelo
6. Interpretação dos resultados
